# simon-writing

김서준(해시드 대표)의 글쓰기 스타일을 분석하고, 그 문체로 에세이를 생성하는 AI 스킬.

## 이건 뭔가

김서준 대표님의 글을 좋아하는 한 사람의 팬심에서 시작된 오마주다.

27편의 에세이와 단편소설을 AI에게 읽히고, "이 사람은 어떻게 이렇게 쓰는 거지?"를 6개의 관점에서 해체했다. 의식의 흐름, 메타포의 구조, 지식의 배치법, 문장의 호흡, 표현의 패턴, 소재를 찾아오는 경로까지. 만들다 보니 흡사 국어 교과서를 쓰는 느낌이었는데 — 그만큼 좋은 글이었다는 뜻이다.

결과물은 두 가지다. 하나는 글쓰기 분석 자료 그 자체(독립적인 교재로 읽힌다). 다른 하나는 그 분석을 바탕으로 AI가 유사한 문체의 에세이를 생성하는 스킬.

## 사용 전에

**이 스킬은 김서준 대표님의 저작물을 분석한 결과물입니다.**
사용 시 원저자의 허락을 받고 사용하시길 권합니다.
분석 대상 원문의 저작권은 김서준 대표님에게 있습니다.

## 제작 과정

| 항목 | 수치 |
|------|------|
| 분석 대상 | 에세이 및 단편소설 27편 |
| 원문 분량 | 약 200,000자 (한국어) |
| 분석 산출물 | 리서치 6편 + 종합 분석 1편 (약 214,000자) |
| 소요 토큰 | 약 300,000 토큰 |
| 소요 시간 | 약 15분 (서브에이전트 3개 병렬) |
| 사용 모델 | Claude Opus 4.6 |
| 제작일 | 2026년 3월 5일 |

## 사용 방법

Claude Code에서 `/simon-writing` 또는 `/simon`을 입력하면 스킬이 실행된다. 일단 설치를 하시라. 설치는 이 주소를 알아서 복사해서 설치하자고 말한다.

```
/simon-writing
```

주제를 입력하면 자동으로:
1. **원문 참조 읽기** — 주제 유형에 맞는 원문 2-3편을 읽어 호흡과 온도를 몸에 익힘
2. **소재 리서치** — 서브에이전트 2-3개가 병렬로 사실, 역사, 먼 소재, 감각적 장면을 수집
3. **중간 노트** — 핵심 질문 도출, 소재 선별, 흐름 스케치
4. **초안 작성** — 김서준식 문체 원칙을 적용한 에세이 생성
5. **탈형식화 퇴고** — 조립 흔적 제거, 감각 삽입, 리듬 파괴와 재구성
6. **최종 출력** — md 파일 저장

2단계와 3단계 사이에 사용자 확인 포인트가 있어 방향을 조율할 수 있다.

결과물 저장 위치는 사용자에게 문의하거나, 현재 폴더의 `/outputs_writing` 폴더에 저장된다.

## 글쓰기 교재로 읽기

이 프로젝트의 진짜 가치는 스킬 자체보다 **분석 자료**에 있을 수 있다.
AI를 쓰지 않더라도, 아래 파일들은 독립적인 글쓰기 교재로 읽힌다.

특히 `분석4-표현의연금술-40선.md`는 "기가 막힌 문장이 어떻게 만들어지는가"를
40개 사례로 역추적한 문서다. 글을 쓰는 사람이라면 한번 읽어볼 만하다.

## 파일 구조

```
.claude/skills/simon-writing/
├── README.md                              ← 이 파일
├── SKILL.md                               ← AI 스킬 정의 (생성 파이프라인 + 문체 원칙)
│
└── references/
    ├── analysis/                           ← 분석 리서치 (7편)
    │   ├── 종합분석-김서준-글쓰기스타일.md    ← 전체 종합
    │   ├── 분석1-의식의흐름-메타포-호흡.md    ← 의식의 흐름, 비유, 문장 리듬
    │   ├── 분석2-지식구조-레퍼런스-메타인지.md ← 지식 배치, 인문학 레퍼런스
    │   ├── 분석3-서사구조-어조-고유성.md      ← 서사 구조, 화자 포지션
    │   ├── 분석4-표현의연금술-40선.md         ← 기가 막힌 표현 40개 + 사고 역추적
    │   ├── 분석5-소재탐색-중간노트.md         ← 소재 인벤토리, 수집 방법론
    │   └── 분석6-파이프라인-프롬프트설계.md    ← 생성 파이프라인, 유형별 프롬프트
    │
    └── posts/                              ← 원문 포스팅 (27편)
        ├── 길바닥에_쓰여진_분필의_기도.md
        ├── 사라져_가는_직함들.md
        ├── 딸깍의_신학.md
        └── ... (총 27편)
```

## 분석 파일 가이드

### 종합분석-김서준-글쓰기스타일.md
27편 전체를 관통하는 종합 분석. 9개 챕터.
- 사유의 건축 (도입-전개-전환-결말 패턴)
- 메타포의 무기고 (비유의 5대 출처, 배치 전략)
- 지식의 직조 (분야 교차, 전문 지식 번역의 3박자)
- 문장의 음악 (호흡, 줌인 효과, 여백, 소리)
- 메타인지적 글쓰기 (사유의 촉발, 전환점, 선제 차단)
- 어조와 화자 ("먼저 본 정찰병")
- DNA 5가지 (비유=논증, 닫지 않는 결말, 기술→존재론, 시간축 조작, 소리가 들리는 문장)
- 프롬프트 재현 가이드 + 함정 목록

### 분석1 — 의식의 흐름, 메타포, 호흡
9편을 대상으로 한 미시 분석. 사유가 어떻게 흘러가는지, 비유가 어떻게 배치되는지, 호흡이 어떻게 조절되는지.

### 분석2 — 지식 구조, 레퍼런스, 메타인지
9편을 대상으로 한 거시 분석. 어떤 분야에서 지식을 끌어오는지, 레퍼런스의 시간적 범위, 자기 사유에 대한 사유.

### 분석3 — 서사 구조, 어조, 고유성
9편을 대상으로 한 서사학적 분석. 화자의 위치, 확신의 강도 조절, AI로 재현할 때의 핵심과 함정.

### 분석4 — 표현의 연금술 40선
**이 파일이 가장 재미있다.** 27편에서 "기가 막히다"고 느낀 표현 40개를 뽑고, 각각에 대해 수사학적 분석과 사고 과정 역추적을 수행. "이 문장이 왜 강한가"를 해부하고, "이 사람의 머릿속에서 어떤 과정을 거쳐 이 문장이 나왔는가"를 추적한다. 7가지 표현 생성 패턴(병치, 강등, 격상, 재정의, 감각화, 기생적 역설, 시간 접기)도 여기서 도출.

### 분석5 — 소재 탐색, 중간 노트
김서준이 소재를 어디서 가져오는지 전수 조사. 역사, 과학, 종교, 일상, 대중문화에서의 소재 출처와 패턴. 글을 쓰기 전 "중간 노트" 구성 템플릿 포함.

### 분석6 — 파이프라인, 프롬프트 설계
분석 결과를 AI 생성 파이프라인으로 변환. 글 유형별(기술 분석, 사회 비평, 개인 에세이, SF) 프롬프트 변주.
