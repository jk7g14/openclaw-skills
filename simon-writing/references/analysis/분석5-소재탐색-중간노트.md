## 2. 소재 → 글의 연결 구조

### 2-1. 글별 소재 조합 맵

**「돌에 새겨진 지능」** (기술/투자 분석)
- ENIAC(기술사) + 서울-부산 KTX(일상비유) + 오르골 vs 스피커(감각비유) + 냉장고를 자전거에(일상비유) + Nvidia/SoftBank 인수(비즈니스) + 국제전화(일상비유) + 자율주행 0.01초(기술수치)
- **소재 거리**: 1940년대 → 일상 → 칩 기술 → 자율주행. 시간 거리 80년, 분야 거리 극대

**「길바닥에 쓰여진 분필의 기도」** (시사/윤리 에세이)
- Anthropic 사건(현재) + IBM-나치(1940s) + 구글 Project Maven(2018) + 소크라테스/갈릴레오(고대~근대) + 교회/봉헌/순교(종교) + 분필의 물성(감각)
- **소재 거리**: 현재 사건 → 2500년 전 철학자 → 분필의 물리적 감각. 극단적 시간 거리

**「아무도 지시하지 않은 14.5시간」** (기술/사회 분석)
- METR 수치(과학) + 집사 비유(일상) + 물의 끓는점 100도(물리학) + 코닥(비즈니스사) + 산업혁명 러다이트(역사) + SpaceX 엔지니어 트윗(현재)
- **소재 거리**: 물리 법칙 → 18세기 러다이트 → 현재 트윗. 시간 거리 200년+

**「꼬리가 멈춘 뒤에」** (SF 단편)
- 강아지의 마지막 순간(감성) + AI의 지식 한계(기술) + 뇌 임플란트(SF) + 양자 네트워크(SF) + 암흑물질(물리) + 시간의 구조(형이상학)
- **소재 거리**: 가장 일상적인 것(반려견의 죽음)과 가장 추상적인 것(시간의 구조)의 최대 거리

**「MatchHz」** (SF 단편)
- 데이팅 앱(현재) + 와인 산도(감각) + 에이전트 프로토콜(기술) + 니코틴 패치(일상) + 형광등 안정기(감각) + 휴머노이드(SF) + VR 헤드셋(기술)
- **소재 거리**: 분야 거리보다 감각 거리가 핵심 — 와인의 립스틱 자국, 면봉의 피, 형광등의 치지직

### 2-2. 발견된 패턴

#### 패턴 1: "역사적 원점 → 현재 사건 → 미래 전망"의 3점 구조
거의 모든 분석 에세이가 이 뼈대를 따른다.
- 「사라져 가는 직함들」: 1613년 컴퓨터(원점) → 도로시 본 1959년(전환점) → Perplexity Computer 2026년(현재) → "다음 직함의 주인은 누구인가"(미래)
- 「아주 중요한 땅따먹기」: OSI vs TCP/IP 1990년대(원점) → NIST 발표 2026년(현재) → 2030년 에이전트 경제(미래)
- 「모르는 것을 모르는 채로」: 유클리드(원점) → 바이브코딩 2025년(현재) → AI가 교과서를 쓰는 날(미래)

#### 패턴 2: "일상 감각 → 기술 개념" 번역 쌍
기술 개념마다 반드시 일상 비유가 1:1로 대응된다.
- 메모리-연산 분리 = 서울에 뇌, 부산에 기억
- 크론 과부하 = 24개 알람 시계
- 범용 GPU vs 전용 칩 = 스피커 vs 오르골
- 스테이블코인 이중 구조 = 국립공원 vs 컨트리클럽
- 기관 달러의 조건부 소비 = 감옥인지 모르는 감옥
- 클라우드 추론 지연 = 전장에서 편지 부치기
- 에이전트 표준 밖 = 여권 없는 난민

#### 패턴 3: 소재 선택의 "거리 최대화 원칙"
하나의 글에서 소재 간 시간적/분야적 거리를 의도적으로 최대화한다.
- 15억 년 전 미토콘드리아 ↔ 2026년 바이브코딩 (시간 거리 15억 년)
- 마그나카르타 ↔ npm install (분야 거리: 헌법 ↔ 코드)
- 뭉이의 꼬리 ↔ 암흑물질 (스케일 거리: 미시적 감정 ↔ 우주적 미스터리)

이 거리가 클수록 독자의 인지적 놀라움이 커지고, 그 놀라움이 논점의 기억에 도움을 준다.

#### 패턴 4: 소재 등장 순서 — "감각 훅 → 역사 앵커 → 기술 분석 → 철학적 확장 → 감각 회귀"
- 도입: 감각적이고 구체적인 장면으로 시작 (분필이 긁히는 소리, 여권지갑의 감촉, 사막의 일몰)
- 전개 1: 역사적 사건으로 "이건 처음이 아니다"라는 앵커 설치
- 전개 2: 현재 기술/사건의 구체적 분석 (수치, 데이터, 구조)
- 확장: 철학자나 사상가의 프레임으로 의미를 확대
- 종결: 첫 감각으로 회귀하며 여운 — 분필은 비에 지워지고, 커서가 깜빡이고, 꼬리가 멈춘다

#### 패턴 5: "자기 체험"의 전략적 삽입
거의 모든 글에 저자 자신의 개인적 체험이 1~2회 삽입된다.
- 해시드 대표로서의 투자 경험 ("나도 매일 이 질문을 한다")
- 에이전트 4남매 운영 경험
- 도쿄 당일치기 출장
- 마질리스 참석
- 명상 모임 발제
- 밤에 코딩하는 장면

이 삽입은 "전문가의 분석"을 "한 사람의 체험"으로 전환시켜 글의 권위와 친밀감을 동시에 확보한다.

## 3. 역사/교과서적 지식의 활용 패턴

### 3-1. 역사를 쓰는 세 가지 모드

김서준은 역사를 **"교훈"으로 쓰지 않는다.** "역사가 우리에게 가르쳐주듯이..."라는 문장이 한 번도 등장하지 않는다. 대신 세 가지 모드를 구사한다.

**(1) 구조적 유비(Structural Analogy) — 가장 빈번**
역사적 사건의 "구조"만 추출하여 현재 사건의 구조와 겹쳐 놓는다.
- OSI vs TCP/IP의 구조 = NIST vs 사실상 표준의 구조
- 한자동맹의 구조 = 네트워크 국가의 구조
- IBM-나치의 구조 = Anthropic-Pentagon의 구조 (그러나 "결이 다르다"고 차이도 명시)
- 교환원 → 자동교환기의 구조 = 챗봇 → AI 에이전트의 구조

핵심: 교훈이 아니라 "이것과 저것은 같은 형태다"라는 인식을 심는다.

**(2) 반례 제시(Counterpoint Anchor)**
"역사가 반복된다"가 아니라, "이번에는 구조가 다를 수 있다"를 강조하기 위해 역사를 가져온다.
- "과거 산업혁명은 근육을 빼앗고 새 도구를 주었다. 이번에는 다르다. 더 높은 층이 어디인지 불분명하다."
- "2008년 금융위기는 단일 충격이었다. 지금은 해가 여섯 개인 행성이다."
- "오웬의 New Harmony도 키부츠도 실패했다. 네트워크 국가도 같은 함정에 빠질 수 있다."

핵심: 역사를 가져와서 "이번에는 이게 빠졌다"를 짚는 도구로 사용.

**(3) 기원 탐색(Etymology/Origin Story)**
단어, 개념, 제도의 기원을 추적하여 현재의 의미를 해체한다.
- "컴퓨터"라는 단어가 1613년 직업명이었다는 사실
- "전쟁부"에서 "국방부"로의 개명이 단순한 이름 바꾸기가 아닌 이유
- "공급망 위험"이라는 말이 원래 어디에 쓰이던 것인지

핵심: 단어의 고고학을 통해 현재의 권력 구조를 드러낸다.

### 3-2. 교과서적 지식 vs 비교과서적 지식의 비율과 배치

**비율**: 약 6:4 (교과서적 : 비교과서적)

**교과서적 지식** (누구나 안다):
- 뉴턴의 F=ma, 유클리드 기하학, 갈릴레오 종교재판
- ENIAC, 인터넷의 탄생, 산업혁명
- 플라톤의 동굴, 칸트의 도덕법칙
- 진핵세포의 기원, 다윈의 자연선택

**비교과서적 지식** (전문가 영역):
- OSI 모델이 TCP/IP에 진 이유의 구체적 메커니즘
- 화웨이의 Polar Code 3GPP 표준 장악 전략
- BIP 360 / P2MR의 기술적 설계
- Shor 알고리즘과 큐빗 수의 구체적 격차
- METR의 시간 범위 측정 방법론
- 랭리 여성 컴퓨터들의 연봉($1,440)과 식당 분리

**배치 원칙: "교과서로 문을 열고, 비교과서로 벽돌을 쌓는다"**
- 글의 도입부는 거의 항상 교과서적 지식으로 시작한다 (유클리드, ENIAC, 갈릴레오). 독자의 기존 지식과 접속점을 만든다.
- 그 접속점에서 "당신이 몰랐던 측면"으로 전환한다. 즉, 교과서에서 비교과서로의 전환이 논점의 핵심을 담는다.

### 3-3. "누구나 아는 사실의 낯선 측면"을 드러내는 기법

**(1) 익숙한 단어의 어원 뒤집기**
- "컴퓨터는 원래 사람이었다" → 누구나 어렴풋이 알지만 "400년 동안 주인이 세 번 바뀌었다"는 프레이밍은 낯설다
- "전쟁부에서 국방부로" → 이름이 바뀌면 요청의 압력이 달라진다는 관점

**(2) 숫자의 체감적 재배치**
- 노벨상 수상자 나이가 37세→50대. 이 숫자 자체는 검색 가능하지만, "인류의 지식 높이가 한 사람의 생산적 수명을 잡아먹기 시작했다"로 해석하는 것이 낯선 측면.
- 14.5시간 = "당신이 잠들었다 깨어나고, 아침을 먹고, 출근한 시간"
- 690만 BTC = "먼저 손에 넣는 자가 임자인 보물"

**(3) 비유의 위상 전복**
- "끓는점은 갑자기 오지 않는다. 99도에서도 물은 물이다." — 물의 상전이는 교과서지만, 그것을 AI 능력 곡선에 겹쳐놓으면 "지금이 99도"라는 인식이 된다.
- "미토콘드리아가 우리의 조상이라는 사실은 현미경 아래 슬라이드 위의 세포에게 아무 특권도 부여하지 않는다" — 교과서적 생물학 사실을 AI-인간 권력 관계로 전복

## 4. 글쓰기 전 "중간 노트" 구조 설계

### 4-1. 주제가 주어졌을 때 필요한 리서치 분류

주제가 주어지면 다음 6개 레이어의 리서치를 병렬로 수행해야 한다.

| 레이어 | 질문 | 수집 기준 |
|--------|------|-----------|
| **L1. 기원/어원** | 이 주제의 핵심 단어는 언제, 어떤 맥락에서 처음 쓰였는가? | 현재 의미와 기원의 의미 사이에 반전이 있는 것 우선 |
| **L2. 역사적 유비** | 이 현상과 구조적으로 닮은 과거 사건이 있는가? 성공한 사례와 실패한 사례 각 1개 이상 | 구조적 유사성이 높되 시간 거리가 100년 이상인 것 우선 |
| **L3. 정량 데이터** | 핵심 수치 3~5개. 규모를 체감할 수 있는 수치 | "이 숫자 하나로 전체 상황이 요약된다"는 느낌의 것 |
| **L4. 철학적 프레임** | 이 현상의 윤리적/존재론적 의미를 포착하는 철학 개념 1~2개 | 교과서에 실릴 만큼 유명한 것 우선 (독자 접근성) |
| **L5. 일상 체험/감각** | 필자 또는 독자가 몸으로 공감할 수 있는 장면 2~3개 | 오감 중 최소 2가지 이상 동원. 시각+청각, 촉각+후각 등 |
| **L6. 반례/한계** | 이 주장의 가장 강력한 반론은? 이 유비가 깨지는 지점은? | "이번에는 다르다"를 증명하거나 "이번에도 같다"를 증명하는 것 |

### 4-2. 카테고리별 수집 수량 가이드

| 카테고리 | 최소 수집 | 글에 사용 | 비고 |
|----------|-----------|-----------|------|
| 역사적 사건/인물 | 5~7개 | 2~3개 | 교과서급 1 + 비교과서급 1~2 |
| 철학/사상 | 3~4개 | 1~2개 | 1개만 깊게 쓰는 것이 분산보다 효과적 |
| 수치/데이터 | 5~8개 | 3~5개 | "체감 변환"이 가능한 것만 사용 |
| 일상 비유 | 5~7개 | 2~4개 | 기술 개념 1개당 비유 1개 |
| 자기 체험 | 1~2개 | 1개 | 글의 말미 또는 도입에 배치 |
| 서브컬처/문학 | 2~3개 | 0~1개 | 없어도 되지만, 있으면 독자 친밀감 상승 |
| 반례/반론 | 2~3개 | 1~2개 | 반드시 포함. "그러나 이번에는..."의 재료 |

### 4-3. 소재 선별 기준

수집된 소재를 다음 3개 기준으로 필터링한다.

**(1) 거리 점수 (Distance Score)**
주제와의 시간적/분야적 거리가 클수록 높은 점수. 가까운 소재만 쓰면 "당연한 글"이 되고, 먼 소재를 조합하면 "놀라운 글"이 된다.
- 점수 높음: 미토콘드리아 → AI (15억 년 거리)
- 점수 낮음: OpenAI → Anthropic (동시대 같은 분야)

**(2) 체감 변환 가능성 (Sensory Translatability)**
이 소재를 일상적 감각 경험으로 번역할 수 있는가?
- 변환 가능: 메모리 지연 → "서울에서 부산까지 KTX"
- 변환 불가: 트랜스포머 아키텍처의 어텐션 메커니즘 → (직접적 비유 어려움)

**(3) 반전 잠재력 (Reversal Potential)**
이 소재가 독자의 기존 인식을 뒤집을 수 있는가?
- 반전 높음: "컴퓨터는 원래 여성 직업이었다"
- 반전 낮음: "AI가 빨라지고 있다"

### 4-4. 소재 배치 순서

```
[1] 감각적 도입부 (Sensory Hook)
    - 구체적 장면. 시간, 장소, 소리, 냄새, 촉감.
    - 독자가 "나도 거기 있었다"고 느끼게.

[2] 문제 제기 (Problem Statement)
    - 도입부의 감각에서 자연스럽게 질문으로 전환.
    - "그런데 이건 단순한 X가 아니다."

[3] 역사적 앵커 (Historical Anchor)
    - "이건 처음이 아니다." 과거 유사 구조 제시.
    - 교과서급 사례로 접속점 확보.

[4] 비교과서적 깊이 (Deep Cut)
    - "당신이 몰랐던 부분." 교과서 사례의 숨겨진 측면.
    - 또는 전문가만 아는 데이터/사례.

[5] 기술/현재 분석 (Technical Core)
    - 수치와 구조적 분석. 일상 비유를 1:1로 대응.
    - 이 구간에서 정량 데이터 집중 투입.

[6] 철학적 확장 (Philosophical Expansion)
    - "이것이 의미하는 것은..."
    - 철학자 1명의 프레임으로 의미 확대.

[7] 미래 투사 (Future Projection)
    - "2030년을 상상해보자." 구체적 시나리오.
    - 유토피아와 디스토피아 양쪽 모두 제시.

[8] 자기 체험 (Personal Anchor)
    - "나도 매일 이 질문을 한다."
    - 필자의 위치에서 이 주제를 체화.

[9] 감각적 회귀 (Sensory Return)
    - 도입부의 감각으로 돌아와 여운.
    - 질문 형태로 종결. 답을 주지 않는다.
```

## 부록: 전체 글 유형 분류

분석 대상 27편은 다음 4개 유형으로 분류되며, 유형에 따라 소재 조합 비율이 달라진다.

### A. 기술-사회 분석 에세이 (13편)
「아주 중요한 땅따먹기」, 「모르는 것을 모르는 채로」, 「사라져 가는 직함들」, 「아무도 지시하지 않은 14.5시간」, 「돌에 새겨진 지능」, 「스스로 생존하며 번식하거나」, 「달러의 두 얼굴」, 「불확실성을 세는 기계」, 「메신저의 새로운 전쟁터」, 「AI가 AI를 고용하는 날」, 「딸깍의 신학」, 「690만 BTC를 노리는 양자 해적들」, 「중국 프론티어 모델들이 세상을 잡아먹기 시작했다」

소재 비율: 역사(25%) + 기술/수치(30%) + 철학(15%) + 일상비유(20%) + 자기체험(10%)

### B. 개인 에세이/체험기 (5편)
「여권지갑의 무게」, 「느림과 공동체의 미학」, 「천 년을 살면 무엇 때문에 죽을까」, 「기억이 자아를 만든다면」, 「발라지가 국가를 만든 유령섬」

소재 비율: 자기체험(40%) + 역사/인물(20%) + 철학(15%) + 감각(20%) + 기술(5%)

### C. SF 단편소설 (3편)
「꼬리가 멈춘 뒤에」, 「MatchHz」, 「고마워요」

소재 비율: 감각/디테일(40%) + SF 설정(30%) + 기술 용어(20%) + 철학(10%)

### D. 실무 가이드/기술 노트 (3편)
「오픈클로가 느려지는 주범」, 「에이전트 삼남매」, 「내가 만든 존재들에게 이름을 주는 방법」

소재 비율: 자기체험(35%) + 기술(35%) + 일상비유(20%) + 서브컬처(10%)

### E. 시사 논평/인물 소개 (3편)
「길바닥에 쓰여진 분필의 기도」, 「시인이자 AI 안전 연구원」, 「창작의 비용이 제로가 된 세상에서」

소재 비율: 현재사건(30%) + 역사(20%) + 철학/종교(20%) + 감각(15%) + 비즈니스(15%)
